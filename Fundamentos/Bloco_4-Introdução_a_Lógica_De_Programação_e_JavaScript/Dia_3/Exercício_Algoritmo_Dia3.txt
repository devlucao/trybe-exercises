Exercício 6
- Declarar a variável (let) numbers para atribuir o array;

- Criar estrutura if/else;
- Inserir primeira condição (if) que é: caso o resto da divisão entre o índice do array numbers por 2 (usando operador módulo'%') seja diferente de zero, o valor deve ser inserido na variável composta(array) ímpar (impar += 1);
- Fechar loop de repetição (for);
- Fechar estrutura condicional (if);
- Criar uma nova estrutura if/else;
- Inserir condição (if) que, caso o array impar seja igual a zero, deve imprimir (console.log) a mensagem "Nenhum número ímpar encontrado";
- Inserir a outra condição (else) que, caso a primeira condição não seja satisfeita, deve-se imprimir (console.log) o array ímpar;
